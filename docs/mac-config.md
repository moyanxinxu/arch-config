# Macos配置

## MacBook信息

MacBook Air A3114

## 安装Brew

```bash
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
```

## Brew常用命令

### 清理所有不再需要的包和依赖项（清理缓存）

```bash
brew cleanup --prune=all
```
